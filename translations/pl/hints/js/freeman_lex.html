function isAcceptablePassword(password: string): boolean {

    const cond1 = password.length> 6;
    var cond2 = /\d/.test(password);
    var cond3 = /\D/.test(password);
    if (password.length> 9) cond2 = cond3 = true; // cond 4
    const cond5 =!(/password/i.test(password));
    
    return!([cond1, cond2, cond3, cond5].includes(false));
}

W tej misji należy dodać do rozwiązania sprawdzenie, czy hasło zawiera co najmniej trzy różne znaki.

Od czego powinienem zacząć?

Najpierw należy przekonwertować password na tablicę za pomocą metody split() z argumentem pustego znaku. 

password.split("")

Następnie proszę przekonwertować tablicę na Set(), aby zachować tylko unikalne znaki. 

new Set(password.split(""))

Ale jak sprawdzić, czy zestaw ma co najmniej trzy znaki?

Zbiory mają taką właściwość jak size. Jest ona podobna do length dla ciągów znaków.

Proszę pokazać mi całą linię.

Proszę spojrzeć: 

const cond6 = new Set(password.split("")).size> 2;

I proszę nie zapomnieć dodać nowego warunku do tablicy w return.

Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Oto ono: 

function isAcceptablePassword(password: string): boolean {

    const cond1 = password.length> 6;
    var cond2 = /\d/.test(password);
    var cond3 = /\D/.test(password);
    if (password.length> 9) cond2 = cond3 = true; // cond 4
    const cond5 =!(/password/i.test(password));
    const cond6 = new Set(password.split("")).size> 2;
    
    return!([cond1, cond2, cond3, cond5, cond6].includes(false));
}

Te wyrażenia regularne są testowane za pomocą metody test(). 

var isAcceptablePassword = pwd =>!pwd.match(/password/i)
&& new Set([...pwd]).size>2
&& (pwd.length> 9 || [/.{7,}/, /\d/, /[a-z]/i].every(x=>x.test(pwd)))
