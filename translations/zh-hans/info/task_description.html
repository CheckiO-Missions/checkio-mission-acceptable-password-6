在这个任务中，你需要创建一个密码验证功能。

验证的条件是：

  • 长度应大于6；
  • 应至少包含一个数字，但不能只包含数字；
  • 有数字或只包含数字不适用于长度大于9的密码。
  • 一个字符串在任何情况下都不应该包含 "密码 "一词；
  • 应至少包含3个不同的（区分大小写）字母（或数字），即使它长于10。

输出：一个逻辑值 {% if is_js %}(boolean){% else %}(bool){% endif %}.
