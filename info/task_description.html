In this mission you need to create a password verification function.

Those are verification conditions:

  • length should be bigger than 6
  • contains at least one digit but it can't be all digits
  • having numbers is not required for password longer than 10
  • string should not contain word "password" in any case
  • consist of 3 different letters

Input: String.

Output: Bool.
